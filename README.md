# AHK Pi-hole
An unofficial Pi-hole GUI written in AutoHotkey


## Features
* Displays 'Total queries', 'Queries blocked', 'Percent blocked' & 'Domains on Blocklist'
* Displays 'Top queries' & 'Top ads'
* Displays Status 'enabled' or 'disabled'
